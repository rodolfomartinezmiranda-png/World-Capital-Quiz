# World Capital Quiz 🌍

A dynamic and interactive web application designed to test and improve users' knowledge of world capitals. Built with a robust Node.js backend, this project demonstrates clean architecture, server-side rendering, and efficient data handling from flat-file systems.

---

## 🚀 Features

* **Dynamic Question Generation:** Randomly serves capital and country quiz combinations to ensure a unique user experience every round.
* **CSV Data Persistence:** Leverages a localized `capitals.csv` file database, showcasing backend parsing and data extraction techniques without the overhead of heavy database engines.
* **Real-time Feedback:** Tracks user scores and provides immediate verification on correct or incorrect answers.
* **Responsive Layout:** Clean user interface designed to work seamlessly across both desktop and mobile views.

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Data Layer:** CSV (Comma-Separated Values) parser
* **Frontend:** HTML5, CSS3, EJS (Embedded JavaScript templates) / JavaScript
